\# Dynamic Insights \& Trends in Retail Sales



\## Project Overview



This project analyzes retail sales data using Python, SQL, and data visualization techniques. The objective is to identify sales trends, profitable products, category performance, and regional sales distribution.



\## Tools Used



\* Python

\* Pandas

\* SQL

\* Matplotlib



\## Dataset



The dataset contains retail sales transactions including:



\* Order ID

\* Order Date

\* Product

\* Category

\* Region

\* Sales

\* Profit

\* Quantity



\## Analysis Performed



1\. Total Sales Analysis

2\. Total Profit Analysis

3\. Sales by Category

4\. Sales by Region

5\. Top Selling Products

6\. Data Visualization using Charts



\## Output



\* Total Sales Generated

\* Total Profit Generated

\* Category-wise Sales Analysis

\* Region-wise Sales Analysis

\* Top 5 Products by Sales

\* Category Chart

\* Top Products Chart



\## Project Files



\* sales\_data.csv

\* retail\_analysis.py

\* retail\_queries.sql

\* category\_chart.png

\* sales\_chart.png



\## Conclusion



The project provides insights into retail business performance by analyzing sales data and visualizing trends. These insights can help organizations make better business decisions and improve profitability.
